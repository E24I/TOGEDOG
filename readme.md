## 🛫 소개
> 🧐 나의 애완동물과의 추억을 남길 공간이 없을까?

- 펫과의 추억을 남길 수 있는 글을 올릴 수 있습니다.
- 피드에 위치를 설정하여 언제 어디서의 추억인지 설정할 수 있습니다.
- 다른 유저들에게 나의 펫을 자랑할 수 있고, 근황을 공유할 수 있습니다.
